Тест сохранения данных при смене языка

Инструкции:

  1. Откройте основное приложение в другой вкладке: http://localhost:8080
  2. Введите свои данные в поля биографии и вакансии
  3. Вернитесь на эту страницу и нажмите "Начать тест"
  4. Тест автоматически проверит сохранение данных при смене языка
